#

#
# Fix: Run the MigrationRunner so the Tenant (and other) module migrations are applied:

#   2. From server/src/MigrationRunner: dotnet run
#      Or with topology: dotnet run -- --topology Monolith
#   3. This creates schema "tenant" and table "tenant.tenants" (and other modules).
#
# Code fix: GlobalExceptionHandlerMiddleware now returns 503 with a clear message when it
